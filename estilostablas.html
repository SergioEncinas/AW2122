Estilos en tablas

En este artículo aprenderemos a aplicar 3 estilos a tablas realizadas en HTML gracias al lenguaje de hoja de estilos CSS



Esta es la tabla HTML de la que partimos:


Primer estilo:

td,th {
padding: 0.5px;
border: 1px solid rgba(0,0,0,0.2);
}

td,th {} --> lo que haya entre los corchetes afectará a los elementos "td" y "th"
padding: 0.5px --> crea un espacio entre el borde y el contenido del elemento de 0.5 píxeles
border 1px solid rgba(0,0,0,0.2) --> añade un borde de un píxel del color indicado

Segundo estilo:

table, td, th {
border: 1px solid #ddd;
text-align: left;
}

table{
border-collapse: collapse;
widht: 100%
}

th, td {
padding: 15px;
}

text-align: left --> Alinea el texto a la izquierda
border-collapse: collapse --> Fusiona los bordes de la tabla
widht: 100% --> Asigna el tamaño completo del elemento

Tercer estilo:

table {
border-collapse: collapse
widht: 100%;
}

th, td
text-align: left;
padding: 8px;
}

tr: hover {background-color: yellow}

border-collapse: collapse --> Fusiona los bordes de la tabla
widht: 100% --> Asigna el tamaño completo del elemento
text-align: left --> Alinea el texto a la izquierda
Padding: 8px -->
